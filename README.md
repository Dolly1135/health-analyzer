AI Health Report Analyzer

A simple AI-based web application that analyzes basic health parameters like Hemoglobin and Blood Sugar levels and provides instant health insights.

#Features
- Analyze Hemoglobin levels
- Analyze Blood Sugar levels
- Detect risks like Anemia and Diabetes
- Instant results

#Tech Stack
- Python
- Flask
- HTML

#Run the Project
pip install flask  
python app.py  
Open http://127.0.0.1:5000
